html5tooltips.js
Tooltips for controls and input fields
with CSS3 animation, no framework required.
Attention to details is a clue to a winning design.
People should have an ability to do hard things in a simple way.
Tooltips are a quite usefull detail when used with buttons and input fields.
It gives an impression of solid UI design.
Customization is important. Styling can be changed. Template can be changed.
Or... you can just pick a color :)
